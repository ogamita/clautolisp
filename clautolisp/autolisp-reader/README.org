* Purpose
~autolisp-reader~ is the reader subsystem for the ~clautolisp~ implementation.

Its responsibility is to accept AutoLISP source text, decode it at the input boundary, tokenize and parse it according to an explicit dialect configuration, and return source-aware reader results without leaking Common Lisp reader semantics into the language.
