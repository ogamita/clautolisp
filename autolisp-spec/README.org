* Purpose
~autolisp-spec~ is the specification subproject of the repository.

Its role is to maintain:

- the main AutoLISP / Visual LISP specification draft,
- the plan for resolving under-specified areas,
- source notes gathered from vendor documentation,
- converted source material used during specification work.

* Status
This is currently the most advanced subproject.

The main specification draft exists, and the remaining work is primarily:

- resolving under-specified semantics,
- validating behavior against real implementations,
- extending coverage and citations.
