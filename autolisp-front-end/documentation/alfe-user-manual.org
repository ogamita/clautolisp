* Introduction

The alfe(1) executable is the unified front-end driving clautolisp (in-process or subprocess) and CAD-resident AutoLISP REPLs (BricsCAD, AutoCAD) via a file-IPC protocol. This manual covers everyday use: actions, dialects, hosts, encodings, init files, the --dry-run plan renderer, and the *AUTOLISP-…* globals available to user code.

This document is a stub seeded by documentation-synchronisation.issue.
Audience: user / integrator.
